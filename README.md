# BookBuddy

Real time full stack chat app using Socket.IO. Allows users to create chat rooms or join existing ones with unique room codes. 

## Tech Stack

+ Python
+ Flask

## Installation

Clone this repository.

#### Install dependencies:

```bash

pip install -r requirements.txt

```

#### To start server:

```bash

python main.py

```

### To visit app:

```bash

http://127.0.0.1:5000/

```
